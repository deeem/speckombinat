# Speckombinat

Wordpress тема для [Державне спеціалізоване підприємство "Чорнобильський спецкомбінат"](http://speckombinat.org.ua/)

Создана с применением:
- _underscores
- twitter bootstrap

Использует Wordpress плагины:
- Advanced Custom Fields
- Custom Post Type UI
